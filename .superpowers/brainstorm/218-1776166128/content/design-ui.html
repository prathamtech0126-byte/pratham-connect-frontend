Key Visual Decisions

  • Filter pill style copied exactly from existing Reports period tabs (same bg-muted/40 container, same active state)
  • Table uses border-collapse with border-slate-200 grid lines — spreadsheet feel
  • Alternating rows: white / bg-slate-50
  • Amount column: right-aligned, monospace font, formatted with toLocaleString('en-IN')
  • Shared Client: green badge for Yes (bg-green-100 text-green-700), red badge for No (bg-red-100 text-red-600)
  • Section sits inside a Card wrapper matching other sections on the Reports page
