How it integrates

When user clicks Custom, a popover opens below the filter bar showing this picker. Clicking Update applies the range and triggers the API fetch. Cancel dismisses without changing the active filter.

Implementation note

This will be built as a self-contained DateRangePicker component inside components/payments/. Presets map directly to the API's filter values — clicking "Today" sets filter=today, "This month" sets filter=monthly, etc. Clicking a custom date range sets filter=custom with startDate + endDate. No external date-picker library needed — built with plain React state + a simple calendar grid.
