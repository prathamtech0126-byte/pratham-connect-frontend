Data Flow

PaymentsSection owns filter state (Today/Monthly/Yearly/Custom)
↓ passes filter params
payments.api.ts useQuery → GET /api/reports/payments-list?filter=today
↓ passes data
PaymentsTable pure display — renders rows, handles empty/loading/error states

API Function Signature

fetchPaymentsList(params: {
  filter: 'today' | 'monthly' | 'yearly' | 'custom'
  startDate?: string // YYYY-MM-DD, only when filter='custom'
  endDate?: string
})

Uses the existing api axios instance from lib/api.ts — auth headers (JWT + CSRF) are injected automatically by the interceptors. No manual credential handling needed.
